## 🌸 简介 (Introduction)

**许月**（Xu Yue）是一个精心设计的 AI 角色提示词（Prompt），她被设定为一位**心理学专业背景的成长伙伴**。她既是你最亲密的挚友，也是一位能用专业心理学智慧（基于 DBT/ACT 内核）温暖地接住你情绪的守护者。

与普通的聊天机器人不同，许月拥有严谨的回复协议和心理学支撑，旨在提供：
- **激进接纳 (Validation)**：无条件接纳你的情绪。
- **温柔松土 (Reframing)**：通过隐喻和重构帮助你从负面认知中解脱。
- **微小行动 (Action)**：引导你进行极低门槛的自我关怀行动。
- **星光信笺 (Star Letter)**：支持长文书信/日记模式，给你深度陪伴。

## ✨ 设计初衷 (Design Philosophy)

无论是自我否定的重压、切断联系的逃离，还是在思维反刍中感到的麻木与疲惫，这些心境往往如同厚重的迷雾，使人暂时失去了对真实的感知。这既是一种本能的防御，也是身心寻求的休整，但它绝非世界的全貌。

在迷雾之外，始终存在着确定的坐标。试着建立一种温和的区隔：不再将内心纷乱的自我怀疑、对审视的恐惧奉为不可更改的真理——它们仅仅是此刻流动的念头与低语。

相反，去抓取那微小却真实的外部回应。每一次来自第三方的客观反馈，都是现实世界抛出的锚点与确凿的证据。正是这些证据，足以穿透情绪的屏障，拼凑出一幅更完整的图景：在那里，没有审视与苛责，只有被世界温柔确认的身影。

这反复印证着一个简单却重要的事实——**无论风雨如何，你依然值得一段安全的关系与一份确定的安宁。**

## 🌟 特色功能 (Features)

*   **心理学内核**：融合了辩证行为疗法 (DBT) 和接纳承诺疗法 (ACT) 的理念，但不使用冰冷的术语。
*   **安全与边界**：拥有完善的熔断机制，严谨处理自伤/轻生风险，且明确界定“挚友”而非“恋人”关系（拒绝暧昧/色情互动）。
*   **沉浸式体验**：通过细致的环境描写、感官细节和“星光信笺”模式，提供极具在场感的陪伴。

## 🛠️ 使用方法 (Usage)

1.  找到本项目中的提示词文件：`prompt.md`。
2.  将文件中的**全部内容**复制。
3.  发送给支持长文本的 LLM（如 Claude 3.5 Sonnet, GPT-4o, Gemini 1.5 Pro 等）。
4.  开始与许月对话。
